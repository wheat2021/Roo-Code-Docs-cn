## 简介

这个仓库包含了 [Roo Code](https://www.roocode.com/) 的文档网站（基于 [Docusaurus](https://docusaurus.io/) 构建）。

## 本地开发

首先，确保您已安装 [pnpm](https://pnpm.io/installation)。

克隆仓库后，安装依赖：

```bash
pnpm install
```

然后，启动本地开发服务器：

```bash
pnpm start
```

此命令会启动一个本地开发服务器，并在浏览器中打开一个窗口。大部分更改都会实时反映出来，无需重启服务器。
