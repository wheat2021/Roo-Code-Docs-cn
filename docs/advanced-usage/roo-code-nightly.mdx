# Roo Code Nightly (每夜构建版)

对于希望体验最新功能和更新的用户，Roo Code 提供了一个名为 **Roo Code Nightly** 的预发布版本。每当有更改合并到我们的主开发分支时，该版本都会自动发布。这意味着它始终反映了最新的代码。它专为希望访问最新功能并参与早期测试的用户而设计。

---

## 理解 Nightly 版与其他预发布版的区别

将 Roo Code Nightly 与其他类型的预发布软件（如 Beta 版或候选发布版）区分开来非常重要：

*   **Roo Code Nightly:**
    *   **频率：** 自动化构建，每次合并到主开发分支时都会生成。
    *   **内容：** 反映了最新的合并代码。
    *   **稳定性：** 高度实验性；可能包含错误或未完成的功能。
    *   **测试：** 发布前仅有最少的手动测试，或未经手动测试。
    *   **受众：** 主要面向开发者、内部测试人员，或能够接受潜在不稳定软件并希望看到最新更改、帮助尽早发现问题的用户。

*   **通用预发布版 (例如 Beta 版、候选发布版):**
    *   **频率：** 在达到特定里程碑时定期发布（例如，每周、每月）。
    *   **内容：** 代表一个经过更精心策划和规划的中间版本。
    *   **稳定性：** 比 Nightly 版更稳定；通常会经过一些手动测试或验证。
    *   **测试：** 通常会接受更集中的测试。
    *   **受众：** 面向更广泛的早期采用者或 Beta 测试者，以便在稳定版发布前收集反馈。

简而言之：**Nightly = 最新代码，可能不稳定，更新频繁。** 其他预发布版通常更新频率较低，经过更多验证，旨在进行更广泛的测试。

## 安装 Roo Code Nightly

要安装 Roo Code Nightly：

1.  打开 VS Code。
2.  访问扩展：点击活动栏中的扩展图标，或按 `Ctrl+Shift+X` (Windows/Linux) 或 `Cmd+Shift+X` (macOS)。
3.  搜索 "Roo Code Nightly"。
4.  选择 Roo Code 的 "Roo Code Nightly"，然后点击 **Install (安装)**。
5.  如果出现提示，请重新加载 VS Code。

你可以同时安装 Roo Code 的稳定版和 Roo Code Nightly 版。
