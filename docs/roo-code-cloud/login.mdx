# 登录

将您的 Roo Code 扩展程序连接到 Roo Code Cloud，以同步您的提示、访问在线任务历史记录，并启用共享和协作功能。

## 入门

要连接到 Roo Code Cloud，请单击 Roo Code 插件右上角的**帐户图标** <Codicon name="account" />。

<img src="/img/login/login.png" alt="Roo Code Cloud 初始连接屏幕" width="600" />

单击**连接**按钮开始身份验证过程。您将被重定向到 Roo Code Cloud 网站。

## 登录选项

您将看到包含多个身份验证选项的登录页面：

<img src="/img/login/login-1.png" alt="Roo Code Cloud 登录页面" width="600" />

选择您偏好的登录方式：
- **GitHub** - 使用您现有的 GitHub 帐户登录
- **Google** - 使用您现有的 Google 帐户登录
- **电子邮件** - 输入您的电子邮件地址以继续进行电子邮件身份验证

## 创建新帐户

如果您还没有帐户，请单击登录页面底部的**注册**链接。

<img src="/img/login/login-2.png" alt="Roo Code Cloud 注册表单" width="600" />

填写注册表单：
- **名字** (可选)
- **姓氏** (可选)
- **电子邮件地址** (必填)
- **密码** (必填)
- 接受服务条款和隐私政策

完成表单后，您将看到一个确认弹出窗口：

<img src="/img/login/login-3.png" alt="帐户创建确认弹出窗口" width="600" />

## IDE 选择

身份验证页面包含一个用于其他 VS Code 兼容 IDE 的选择器：

<img src="/img/login/login-4.png" alt="VS Code 分支的 IDE 选择器" width="600" />

这允许您将 Roo Code Cloud 连接到各种 VS Code 分支和兼容的编辑器。

## 身份验证完成

身份验证完成后，您将被重定向回您的 IDE，在那里您将看到已连接的帐户：

<img src="/img/login/login-5.png" alt="成功连接 Roo Code Cloud 帐户" width="600" />

在此屏幕上，您可以：
- **访问 Roo Code Cloud** - 单击以打开 Web 仪表板
- **注销** - 从此 IDE 断开您的帐户
- **完成** - 关闭帐户面板并返回正常的 Roo Code 使用

您的 Roo Code 扩展程序现已连接到 Roo Code Cloud，并准备好同步您的数据和启用协作功能。
