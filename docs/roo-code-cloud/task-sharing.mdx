# 任务分享

通过创建可分享的链接，将您的 Roo Code 任务设为公开，与他人分享和协作。

:::note 需要登录
您必须登录到 Roo Code Cloud 才能分享任务。如果您尚未关联您的账户，请先按照[登录说明](/roo-code-cloud/login)进行操作。
:::

<img src="/img/task-sharing/task-sharing-1.gif" alt="Roo Code 任务分享演示" width="600" />

<br />

## 在 Roo Code 内部进行分享

### 步骤 1：访问分享选项

在任何活动任务视图中，找到任务标题右侧的**链接图标** <Codicon name="link" />。

<img src="/img/task-sharing/task-sharing.png" alt="活动任务视图中的链接图标" width="600" />

### 步骤 2：选择公开分享

点击链接图标打开分享选项，然后选择**公开分享**。

<img src="/img/task-sharing/task-sharing-1.png" alt="公开分享选项" width="400" />

此选项允许任何拥有链接的人访问您的任务。

### 步骤 3：链接已创建

系统将生成一个公开链接，并自动将其复制到您的剪贴板。

<img src="/img/task-sharing/task-sharing-2.png" alt="公开链接已复制确认" width="400" />

您现在可以将此链接分享给团队成员、协作者或任何需要查看您任务及其对话历史的人。

## 从仪表板分享

### 步骤 1：在仪表板中访问任务

导航到您的 [Roo Code Cloud 仪表板](/roo-code-cloud/dashboard)，然后点击您想要分享的任务以打开详细的任务弹窗。

<img src="/img/task-sharing/task-sharing-3.png" alt="带有分享按钮的仪表板任务弹窗" width="600" />

### 步骤 2：打开分享弹窗

在详细的任务弹窗中，点击**分享**按钮以打开分享选项。

<img src="/img/task-sharing/task-sharing-4.png" alt="分享任务弹窗" width="600" />

该弹窗将显示创建公开链接的选项，任何人都可以访问，并提示链接将在 30 天后过期。

### 步骤 3：创建和管理分享链接

点击**创建分享链接**以生成一个新的公开链接。创建后，您将看到确认界面：

<img src="/img/task-sharing/task-sharing-5.png" alt="分享链接已创建确认" width="600" />

在此界面上，您可以：
- **复制链接** - 将生成的分享链接复制到剪贴板
- **创建另一个链接** - 为同一任务生成额外的分享链接
- **查看以前的链接** - 查看和管理此任务以前创建的链接
- **删除链接** - 使用删除图标移除分享链接

仪表板分享方法提供了更高级的链接管理功能，包括为每个任务创建多个链接以及跟踪以前生成的链接。
