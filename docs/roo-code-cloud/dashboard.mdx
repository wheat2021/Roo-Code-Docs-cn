# 控制面板

Roo Code Cloud 控制面板提供了一个集中的位置来管理您的帐户、查看任务历史记录以及访问共享和协作功能。

## 访问控制面板

有两种方法可以访问您的 Roo Code Cloud 控制面板：

### 方法一：通过 Roo Code 插件

登录后，单击 Roo Code 插件右上角的 **帐户图标** <Codicon name="account" />，然后单击 **访问 Roo Code Cloud**。

<img src="/img/login/login-5.png" alt="Access dashboard from Roo Code plugin" width="600" />

### 方法二：直接通过网页访问

直接导航到 Roo Code Cloud Web 应用程序：

**https://app.roocode.com/**

这使您可以从任何 Web 浏览器访问您的控制面板，即使在不使用 Roo Code 扩展程序时也是如此。

## 控制面板功能

控制面板提供以下访问权限：

- **任务历史** - 查看和管理您的在线任务历史记录
- **帐户设置** - 管理您的个人资料和偏好设置
- **共享与协作** - 访问共享任务和协作功能
- **使用指标** - 监控您的任务、令牌和基于成本的使用情况

您的控制面板与您的 Roo Code 扩展程序同步，以提供您的编码助手活动和协作工作流程的全面视图。
