Roo Code 是一款 VS Code 扩展，可将 AI 驱动的编码辅助功能直接带到您的编辑器中。请使用以下方法之一进行安装：
1. **VS Code 应用商店 (推荐)** - 适用于标准 VS Code 和 Cursor 用户的最快方法
2. **Open VSX Registry** - 适用于像 VSCodium 这样与 VS Code 兼容的编辑器

---

## VS Code 应用商店

1. 打开 VS Code
2. 访问扩展：单击活动栏中的扩展图标或按 `Ctrl+Shift+X` (Windows/Linux) 或 `Cmd+Shift+X` (macOS)
3. 搜索 "Roo Code"
4. 选择由 RooVeterinaryInc 发布的 "Roo Code" 并点击 **安装**
5. 如果出现提示，请重新加载 VS Code

安装后，在活动栏中找到 Roo Code 图标 (<KangarooIcon />) 以打开 Roo Code 面板。

## Open VSX Registry

对于无法访问应用商店的 VS Code 兼容编辑器（如 VSCodium 和 Windsurf）：

1. 打开您的编辑器
2. 访问扩展视图
3. 搜索 "Roo Code"
4. 选择由 RooVeterinaryInc 发布的 "Roo Code" 并点击 **安装**
5. 如果出现提示，请重新加载

## 从 VSIX 手动安装

如果您希望直接下载并安装 VSIX 文件：

1. **下载 VSIX 文件：**
   * 在 [Roo Code GitHub 发布页面](https://github.com/RooCodeInc/Roo-Code/releases)上查找官方版本
   * 从最新版本下载 `.vsix` 文件

2. **在 VS Code 中安装：**
   * 打开 VS Code
   * 访问扩展视图
   * 单击扩展视图中的 "..." 菜单
   * 选择 "从 VSIX 安装..."
   * 浏览并选择您下载的 `.vsix` 文件

## 问题排查

<img src="/img/installing/installing-4.png" alt="显示用于故障排查的 Roo Code 日志的 VS Code 输出面板" width="100%" />
*显示用于故障排查的 Roo Code 日志的 VS Code 输出面板*

**扩展不可见**
* 重启 VS Code
* 验证 Roo Code 是否在扩展中列出并已启用
* 尝试禁用并重新启用
* 检查输出面板中的错误（查看 → 输出，选择 "Roo Code"）

**安装问题**
* 确保网络连接稳定
* 验证 VS Code 版本是否为 1.84.0 或更高版本
* 如果无法访问 VS Code 应用商店，请尝试 Open VSX Registry 方法
