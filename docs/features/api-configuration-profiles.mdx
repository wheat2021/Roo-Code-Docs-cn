# API 配置方案

API 配置方案允许您创建并在不同的 AI 设置组之间切换。每个方案可以为每种模式设置不同的配置，让您可以根据手头的任务优化您的体验。

:::info
拥有多个配置方案可以让您快速在不同的 AI 提供商、模型和设置之间切换，而无需在每次想更改设置时都重新配置所有内容。
:::

## 工作原理

配置方案可以拥有自己的：
- API 提供商（OpenAI、Anthropic、OpenRouter、Glama 等）
- API 密钥和身份验证详情
- 模型选择（o3-mini-high、Claude 3.7 Sonnet、DeepSeek R1 等）
- 用于控制响应随机性的[温度设置](/features/model-temperature)
- 思考预算
- 特定于提供商的设置
- [差异编辑配置](/features/fast-edits)
- 速率限制设置

请注意，可用设置因提供商和模型而异。每个提供商提供不同的配置选项，即使在同一提供商内，不同的模型也可能支持不同的参数范围或功能。

## 创建和管理方案

### 创建方案

1. 点击齿轮图标 <Codicon name="gear" /> → 提供商 打开设置
2. 点击方案选择器旁边的“+”按钮

   <img src="/img/api-configuration-profiles/api-configuration-profiles-1.png" alt="带加号按钮的方案选择器" width="550" />
3. 为您的新方案输入一个名称
   
   <img src="/img/api-configuration-profiles/api-configuration-profiles.png" alt="创建新方案对话框" width="550" />
4. 配置方案设置：
   - 选择您的 API 提供商
      
      <img src="/img/api-configuration-profiles/api-configuration-profiles-2.png" alt="提供商选择下拉菜单" width="550" />
   - 输入 API 密钥
   
      <img src="/img/api-configuration-profiles/api-configuration-profiles-3.png" alt="API 密钥输入字段" width="550" />
   - 选择一个模型
   
      <img src="/img/api-configuration-profiles/api-configuration-profiles-8.png" alt="模型选择界面" width="550" />
   - 为此方案配置**速率限制**：
     - **默认为 0（禁用），适用于大多数用户。** 如果需要，您可以设置*此方案*的 API 请求之间的最短时间（以秒为单位），以帮助管理成本或避免提供商的速率限制。
     - 值为 0 表示禁用速率限制（默认）。
     - 使用其他方案的请求遵循其各自的速率限制。

         <img src="/img/api-configuration-profiles/api-configuration-profiles-12.png" alt="API 方案设置中的速率限制滑块控件" width="550" />
   - 调整模型参数（如[温度](/features/model-temperature)）
   
### 切换方案

有两种方式可以切换方案：
1. 从设置面板：从下拉菜单中选择一个不同的方案

   <img src="/img/api-configuration-profiles/api-configuration-profiles-7.png" alt="设置中的方案选择下拉菜单" width="550" />
2. 在聊天期间：访问聊天界面中的 API 配置下拉菜单

   <img src="/img/api-configuration-profiles/api-configuration-profiles-6.png" alt="聊天界面中的 API 配置下拉菜单" width="550" />
### 置顶和排序方案

API 配置下拉菜单现在支持置顶您最喜欢的方案以便更快访问：

1. 将鼠标悬停在下拉菜单中的任何方案上以显示置顶图标
2. 点击置顶图标将方案添加到您的置顶列表
3. 置顶的方案会出现在下拉菜单的顶部，按字母顺序排序
4. 未置顶的方案会出现在分隔符下方，也按字母顺序排序
5. 您可以再次点击相同的图标来取消置顶方案

<img src="/img/api-configuration-profiles/api-configuration-profiles-4.png" alt="置顶 API 配置方案" width="550" />

此功能使在常用方案之间导航更加容易，尤其是在您有许多配置时。


### 编辑和删除方案

<img src="/img/api-configuration-profiles/api-configuration-profiles-10.png" alt="方案编辑界面" width="550" />
- 在设置中选择方案以修改任何设置
- 点击铅笔图标重命名方案
- 点击垃圾桶图标删除方案（您不能删除唯一剩下的方案）

## 将方案链接到模式
在 <Codicon name="notebook" /> 提示选项卡中，您可以将特定的配置方案与每个模式明确关联。系统还会自动记住您上次与每个模式一起使用的方案，使您的工作流程更高效。
<img src="/img/api-configuration-profiles/api-configuration-profiles-11.png" alt="提示选项卡中的方案-模式关联界面" width="550" />

## 安全说明

## 相关功能

- 与您创建的[自定义模式](/features/custom-modes)配合使用
- 与[本地模型](/advanced-usage/local-models)集成以进行离线工作
- 支持每种模式的[温度设置](/features/model-temperature)
- 支持每个方案的速率限制（在此处配置）和通用的[使用情况跟踪/成本信息](/advanced-usage/rate-limits-costs)
- 支持[差异/快速编辑](/features/fast-edits)以实现量身定制的代码编辑行为。
